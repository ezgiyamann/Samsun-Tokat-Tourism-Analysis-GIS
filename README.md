# GIS-Based Tourism Investment Analysis
This project evaluates the tourism potential of Samsun and Tokat provinces using spatial data analysis.

## Objectives
- Compare provinces based on the number of hotels and tourist attractions.
- Analyze domestic and foreign tourist statistics using spatial queries.

## Methodology
- **Data Integration:** Aggregated regional tourism data into a spatial database.
- **Spatial Queries:** Utilized SQL for data filtering and comparative calculations.
- **Visualization:** Created thematic maps in QGIS to highlight investment-friendly zones.
